# My VUE ToDo app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
Backend:
```
npx nodemon index.js
```
Frontend:
```
npm run serve
```

### Production running
Build VUE app:
```
npm run build
```
Run the server:
```
node index.js
```
